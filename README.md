Falar sobre o projeto, sobre como usar o vuetify apartir de um projeto vue:
- npm create vue@latest
- cd vuetify
- npm install
- npm i vuetify
Falar da configuração no @vite.config.js e @eslint.config.js

Uso e instalação de fontes: 
- npm i --save-dev unplugin-fonts
- npm i @fontsource/roboto

Falar sobre a criação dos componentes com vuetify, de acordo com codigo do projeto (componentes do vuetify começam com prefixo "v-").

Adicionar unplugin-auto-import
- npm i -D unplugin-auto-import
Icones: npm install @mdi/font -D
Adicionar npm i -D unplugin-vue-components
